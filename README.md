# gamz-voice

**[ What is this ]**
- Previews the voice range when holding down **SHIFT + H**
- [Check this video](https://streamable.com/138qa)

**[ INSTALLATION ]**
- Download the script and put it in your resources folder
- Add "start gamz-voice" in your server.cfg
